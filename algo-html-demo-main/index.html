Smart Contracts

Limit Order

A Limit Order is a Smart Contract that only pays out funds when a limit amount is reached in the contract logic.

Hash Time Lock Contract

A Hash Time Lock Contract is a Smart Contract that only pays out funds when a certain "preimage" is supplied to it. Think of a preimage as a password or passphrase you need to unlock the funds.

Split Contract

A Split Contract is a Smart Contract that can be used to disburse a payment amount to two wallets in defined proportions.
